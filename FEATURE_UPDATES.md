## [2024-11-XX] - Lesson 1 Release

### Added
- Initial lesson on tokenization, vocabulary, encoding, and decoding
- Jupyter notebook: `LLM_|_Basic_|_L1.ipynb`
- Jupyter notebook: `LLM_|_Basic_|_L2.ipynb`
- README with lesson overview and setup instructions
- Example demonstrating text tokenization using regex
- Vocabulary builder for token-to-ID mapping
- Encode and decode functions for text processing

### Notes
- Based on "Build a Large Language Model" by Sebastian Raschka
- Uses only Python standard library (no external dependencies)
